## Tech Stack

- [React](https://react.dev/) 19
- [Tailwind CSS](https://tailwindcss.com/) 4
- [Vite](https://vite.dev/)

## Development

```bash
git clone https://github.com/azora-tech/azora-lang-website.git
cd azora-lang-website
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
